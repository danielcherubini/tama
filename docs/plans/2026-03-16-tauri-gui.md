# Tauri GUI Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.
> **Status:** TODO - GUI feature, not yet implemented

**Goal:** Build a lightweight desktop GUI using Tauri for non-CLI users to manage koji profiles, services, and models.

**Architecture:** New `koji-gui` crate using Tauri v2. The Rust backend exposes `koji-core` operations as Tauri commands (IPC). The frontend is a minimal web UI (vanilla HTML/CSS/JS or a lightweight framework like Preact). Features: profile management, service start/stop, model pull, status dashboard, log viewer. Reuses all existing `koji-core` logic — the GUI is purely a frontend.

**Tech Stack:** Rust, Tauri v2, `koji-core`, HTML/CSS/JS (or Preact), `serde_json` for IPC

- [ ] **Step 1: Write commands.rs with IPC handlers**

```rust
// crates/koji-gui/src/commands.rs
use koji_core::config::Config;
use serde::Serialize;

#[derive(Serialize)]
pub struct ProfileInfo {
    pub name: String,
    pub backend: String,
    pub use_case: Option<String>,
    pub model: Option<String>,
    pub quant: Option<String>,
    pub service_status: String,
}

#[tauri::command]
pub async fn list_profiles() -> Result<Vec<ProfileInfo>, String> {
    let config = Config::load().map_err(|e| e.to_string())?;
    let mut profiles = Vec::new();

    for (name, profile) in &config.profiles {
        let service_name = Config::service_name(name);
        let status = {
            #[cfg(target_os = "windows")]
            { koji_core::platform::windows::query_service(&service_name)
                .unwrap_or_else(|_| "UNKNOWN".to_string()) }
            #[cfg(target_os = "linux")]
            { koji_core::platform::linux::query_service(&service_name)
                .unwrap_or_else(|_| "UNKNOWN".to_string()) }
            #[cfg(not(any(target_os = "windows", target_os = "linux")))]
            { let _ = &service_name; "N/A".to_string() }
        };

        profiles.push(ProfileInfo {
            name: name.clone(),
            backend: profile.backend.clone(),
            use_case: profile.use_case.as_ref().map(|u| u.to_string()),
            model: profile.model.clone(),
            quant: profile.quant.clone(),
            service_status: status,
        });
    }

    Ok(profiles)
}

#[tauri::command]
pub async fn start_profile(name: String) -> Result<String, String> {
    let service_name = Config::service_name(&name);
    #[cfg(target_os = "windows")]
    koji_core::platform::windows::start_service(&service_name).map_err(|e| e.to_string())?;
    #[cfg(target_os = "linux")]
    koji_core::platform::linux::start_service(&service_name).map_err(|e| e.to_string())?;
    Ok(format!("Started {}", name))
}

#[tauri::command]
pub async fn stop_profile(name: String) -> Result<String, String> {
    let service_name = Config::service_name(&name);
    #[cfg(target_os = "windows")]
    koji_core::platform::windows::stop_service(&service_name).map_err(|e| e.to_string())?;
    #[cfg(target_os = "linux")]
    koji_core::platform::linux::stop_service(&service_name).map_err(|e| e.to_string())?;
    Ok(format!("Stopped {}", name))
}

#[tauri::command]
pub async fn list_models() -> Result<Vec<String>, String> {
    let config = Config::load().map_err(|e| e.to_string())?;
    let models_dir = config.models_dir().map_err(|e| e.to_string())?;
    let registry = koji_core::models::ModelRegistry::new(models_dir);
    let models = registry.scan().map_err(|e| e.to_string())?;
    Ok(models.iter().map(|m| m.id.clone()).collect())
}
```

- [ ] **Step 2: Write app.js**

```javascript
const { invoke } = window.__TAURI__.core;

async function loadProfiles() {
    const profiles = await invoke('list_profiles');
    const container = document.getElementById('profile-list');
    container.innerHTML = profiles.map(p => `
        <div class="profile-card">
            <h3>${p.name}</h3>
            <p>Backend: ${p.backend} | Use case: ${p.use_case || 'none'}</p>
            <p>Status: <span class="status-${p.service_status.toLowerCase()}">${p.service_status}</span></p>
            ${p.model ? `<p>Model: ${p.model} / ${p.quant || '?'}</p>` : ''}
            <div class="actions">
                <button onclick="startProfile('${p.name}')">Start</button>
                <button onclick="stopProfile('${p.name}')">Stop</button>
            </div>
        </div>
    `).join('');
}

async function startProfile(name) {
    await invoke('start_profile', { name });
    loadProfiles();
}

async function stopProfile(name) {
    await invoke('stop_profile', { name });
    loadProfiles();
}

async function loadModels() {
    const models = await invoke('list_models');
    const container = document.getElementById('model-list');
    container.innerHTML = models.map(m => `<div class="model-item">${m}</div>`).join('');
}

// Initial load
loadProfiles();
loadModels();

// Auto-refresh every 5 seconds
setInterval(loadProfiles, 5000);
```
